# SCT_DA_4 - Marketing Campaign Performance Analysis Dashboard
SkillCraft Technology Data Analyst Internship - Task 04
# Project Overview

This project was completed as part of the SkillCraft Technology Data Analyst Internship (SCT_DA_4).

The objective of this task was to perform Exploratory Data Analysis (EDA) on a marketing campaign dataset, analyze campaign performance, evaluate Return on Investment (ROI), conversion rates, acquisition costs, and provide business recommendations based on data-driven insights using Power BI.

# Objectives
Perform Exploratory Data Analysis (EDA)
Analyze marketing campaign effectiveness
Evaluate ROI across marketing channels
Compare acquisition costs and conversion rates
Identify high-performing campaign types
Generate business insights and recommendations

# Key Insights
Analyzed over 200,000 marketing campaign records.
Compared ROI performance across multiple marketing channels.
Evaluated acquisition costs and conversion rates.
Identified campaign types contributing to higher returns.
Generated actionable business recommendations for budget allocation.
# Business Recommendations
Increase investment in channels with strong ROI.
Optimize channels with higher acquisition costs.
Focus on campaign types delivering consistent performance.
Use data-driven insights to improve future marketing strategies.
# Project Outcome

The dashboard provides a comprehensive overview of marketing campaign performance and supports strategic decision-making through interactive visualizations and business insights.
